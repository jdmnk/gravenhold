# Gravenhold UI Assets

These files are legacy display-only UI materials from the first UI pass. The
active in-game frames now use `public/assets/pixel-pack`, specifically Kenney's
UI Pack - Pixel Adventure. These files remain available as fallback/reference
assets, but runtime CSS should prefer the pixel pack.

UI assets must not carry gameplay rules, stats, item metadata, or chain
assumptions.
